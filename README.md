# IntelliJ QuakeC

## Development

1. `git clone`
2. `gradle idea`
3. Open with IDEA
4. Generate JFlex Lexer with [Grammar-Kit](https://github.com/JetBrains/Grammar-Kit)
5. `gradle runIdea`
